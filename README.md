# EquipmentTracker

Система учета оборудования и его обслуживания

## Описание

EquipmentTracker - это приложение для учета оборудования, его перемещений между отделами и сотрудниками, планирования технического обслуживания и генерации отчетов.

## Технологии

- Python 3.8+
- PyQt6
- SQLite

## Установка

1. Убедитесь, что у вас установлен Python 3.8 или выше
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Функциональность

### Реестр оборудования
- Добавление, редактирование и удаление оборудования
- Поиск по инвентарному номеру (< 1 сек)
- Учет характеристик: инвентарный номер, наименование, категория, дата покупки, цена, местоположение, статус

### История перемещений
- Учет назначений оборудования сотрудникам и отделам
- История всех перемещений
- Автоматическое обновление текущего местоположения

### Техническое обслуживание
- Планирование и учет ТО
- Различные типы обслуживания (плановое, внеплановое, ремонт и т.д.)
- Учет стоимости обслуживания

### Отчеты
- Отчет по амортизации оборудования
- Отчет по стоимости содержания
- Отчет по техническому обслуживанию (формирование < 5 сек)

## Структура базы данных

### Таблица equipment
- id - уникальный идентификатор
- inventory_number - инвентарный номер (уникальный)
- name - наименование
- category - категория
- purchase_date - дата покупки
- purchase_price - цена покупки
- current_location - текущее местоположение
- status - статус (active, in_repair, written_off, reserved)

### Таблица maintenance
- id - уникальный идентификатор
- equipment_id - ссылка на оборудование
- maintenance_date - дата обслуживания
- type - тип обслуживания
- cost - стоимость
- description - описание

### Таблица assignments
- id - уникальный идентификатор
- equipment_id - ссылка на оборудование
- assigned_to - кому назначено
- department - отдел
- start_date - дата начала
- end_date - дата окончания (NULL для текущего назначения)

## Критерии приемки

✅ Поиск оборудования по инвентарному номеру < 1 сек  
✅ Формирование отчета по ТО за 5 сек  
✅ Учет статуса оборудования в реальном времени
